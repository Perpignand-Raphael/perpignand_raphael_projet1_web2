
# Documentation

## Structure HTML
- Il y a un header qui contient un logo avec un sous-titre du nom du site, une navigation qui contient chacune des sections du site web (Programtion, Horaire, Artistes, Billets, Partenaires, Témoignages, Contact) et un bouton pour obtenir des billets pour le festival.
## Composante réutilisables
- Ils y a plusieurs boutons obtenez vos billets qui pourrait être stylisé pareil. 
- Les liens dans le header et le footer sont aussi stylisé de la même.
- Les boutons sur la section Lumière urbaine et Contact sont les mêmes.
- Tous les titres sont stylisé de la même façon. 
- Dans la section Programmation et Artistes, les images sont stylisé de la même façon.
- Dans la section Horaire et Témoignage, les articles sont designé de la même façon.
- Dans la section Billets, les icons stylisé de la même façon.
- Dans la section Programmation, il y a un petit tag dans chaques images.
## Nomenclature des classes
- Je vais utiliser la nomenclature BEM pour nommer mes variables.
## Système de tokens
- Je vais utliser les variables de couleurs qui sont déja dans Figma.
- Pour la taille du font, j'ai trouvé 5 tailles qui revient souvent dans le site.
- Pour le padding, j'ai aussi trouvé plusieurs tailles qui revient.
